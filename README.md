# 🧑🏾‍💻 Jokempô

Jogo clássico de Pedra, Papel e Tesoura

## 📝 Descrição

Neste código eu quis fazer o jogo clássico de Pedra, Papel e Tesoura. Aonde você pode escolher se vai jogar só uma rodada ou vai jogar mais rodadas.

## 💻 Funcionalidades

- Verificação de vitória
- Jogo sem e com rodadas
- Placar

## 🗒️ Conceitos Aplicados

- Modularização de código
- Organização por múltiplos arquivos
- Estrutura de execução com `main`
- Estruturas de Condicionais (`if`, `elif`, `else`)
- Estrutas de Repetição `While`
- Operadores Lógicos (`and`, `or`)
- Manipulação de Listas
- Controle de Fluxo com contador
- Função com Parâmetros
- Tratamento de Excessões (`try/except`)
- Uso de Biblioteca padrão (`random`, `time`)
- Implementação de lógica de jogo e sistema de pontuação
